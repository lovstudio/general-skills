# lovstudio-bp-outline

![Version](https://img.shields.io/badge/version-0.1.0-CC785C)

把项目资料整理成投资人能看懂的产品定义、证据账本和 12–15 页 BP 大纲。

## 安装

```bash
npx lovstudio skills add bp-outline -g -y
```

## 使用

```text
$lovstudio-bp-outline 根据当前仓库和用户访谈写 BP 大纲
$lovstudio-bp-outline 重写这份大纲，先不要做 PPT
```

## 交付物

- `brief.md`
- `evidence-ledger.md`
- `outline.md`
- 待补证据清单

大纲确认后，可交给 `lovstudio-bp-deck` 继续制作 PPT。
